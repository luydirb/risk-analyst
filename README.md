# Carregar Dados: 
Os dados são carregados e a coluna transaction_date é convertida para o formato datetime.

# Calcular Média: 
A média dos valores das transações é calculada.

# Identificar Dispositivos Suspeitos: 
Dispositivos usados por múltiplos usuários são identificados.

# Adicionar Diferença de Tempo: 
A diferença de tempo entre transações para cada usuário é calculada.

# Função Antifraude: 
A função simple_anti_fraud verifica a transação com base nas regras definidas (valor alto, dispositivo suspeito e transações frequentes).

# Aplicar Função: 
A função é aplicada a todas as transações e os resultados são armazenados.

# Salvar Resultados: 
Os resultados são salvos em um arquivo Excel.
